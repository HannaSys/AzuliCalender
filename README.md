# Einsatzplanungstool für Auszubildende (ASP.NET Webapplikation)

Ein webbasiertes Einsatzplanungstool zur effizienten Verwaltung von Ausbildungsplänen, Abwesenheiten und Reportings.  
Die Anwendung ersetzt die bisherige manuelle Excel‑Planung durch eine zentrale, datenbankgestützte Lösung.

---

## 🚀 Projektüberblick

Das Projekt umfasst die Entwicklung einer ASP.NET‑Webanwendung zur Einsatzplanung von Auszubildenden.  
Es beinhaltet:

- zentrale Datenhaltung statt Excel  
- automatisierte Berücksichtigung von Feiertagen & Wochenenden  
- Verwaltung von Abwesenheiten (Urlaub, Berufsschule, Prüfungen, Praxis)  
- Reporting für Ausbildungsbeauftragte  
- Benutzerrollen für Ausbilder & Auszubildende  

---
