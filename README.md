# AzuliCalender
Entwicklung einer datenbankgestützten ASP.NET-Webanwendung in C# zur Einsatzplanung von Auszubildenden. Vereinfachen der Einsatzplanung unter Berücksichtigung der Berufszweige, Erstellung eines Reportings für Ausbildungsbeauftragte und Implementierung einer Funktion zur selbstständigen Verwaltung von Urlaub und Abwesenheiten durch die Azubis.
